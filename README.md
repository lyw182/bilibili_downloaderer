## 使用
需求 python>=3.12 pip
1. 以单独项目使用 
   1. 安装依赖
   `pip install -r requirement.txt`
   2. 启动
   `python main.py`
2. 以方法库使用
   1. 在所需项目中导入python_bili
   `from python_bili import *` or `import python_bili`
